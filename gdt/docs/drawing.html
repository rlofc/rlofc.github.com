class gdt::asset¶

An asset is an almost drawable 3D entity you load from a file, holding the required surface data you need to provide your rendering pipeline.

Why almost?

In GDT, an asset object contains the 3D vertices information of a model, without any material properties or texture map data. These attributes are stored in gdt::material objects or being set directly using gdt::pipeline instances.

Another part missing from gdt::asset objects in the specific instance information about the position, rotation and scale of the model in the 3D world containing it. This information is being stored in the gdt::instance or gdt::instances template that can contain assets or other type of GDT entities.

Inherits from gdt::is_entity< ACTUAL >, gdt::is_drivable< ACTUAL >
